# 소수
약수가 1과 자기 자신밖에 없는 수

**에라토스테네스의 체**를 활용한 소수 찾기



#### 2보다 큰 자연수 N의 소수 찾기

``` swift
func getPrimeNumbers(N: Int) -> [Int] {
    var primes = [Int]()
    var isPrime = [Bool](repeating: true, count: N + 1)
    
    for i in 2...N {
        if isPrime[i] {
            primes.append(i)
            
            for j in stride(from: i, to: N + 1, by: i*i) {
                isPrime[j] = false
            }
        }
    }
    
    return primes
}

getPrimeNumbers(N: 11) // returns [2, 3, 4, 5, 7, 8, 9, 11]
```
